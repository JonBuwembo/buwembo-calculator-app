# Calculator Web App

A simple calculator built with **React** and **mathjs** for evaluating expressions. Supports both **button clicks** and **keyboard input** for a seamless user experience.

---

## Features

- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Decimal numbers
- Clear (`C`) and Backspace functionality
- Keyboard support:
  - Numbers: `0-9`
  - Operators: `+ - * /`
  - `Enter` for equals
  - `Backspace` to delete last character
  - `Esc` to clear input
- Error handling for invalid expressions

---

## Technologies Used

- React
- JavaScript (ES6+)
- CSS for styling
- [mathjs](https://www.npmjs.com/package/mathjs) library for safe mathematical evaluation
